SOCIALHUB - QUICK DEPLOYMENT REFERENCE
=====================================

1. PREREQUISITES
   ✓ Node.js 18+
   ✓ PostgreSQL 14+
   ✓ Redis 7+
   ✓ Domain name
   ✓ SSL certificate

2. ENVIRONMENT VARIABLES (.env)
   DATABASE_URL=postgresql://user:pass@host/socialhub
   REDIS_HOST=localhost
   REDIS_PORT=6379
   JWT_SECRET=your-secret-key
   API_URL=https://yourdomain.com
   [OAuth credentials for 5 platforms]

3. QUICK SETUP
   $ npm install
   $ npm run db:setup
   $ npm run build

4. START SERVERS
   Production:  $ npm start
   Development: $ npm run dev

5. NGINX CONFIG
   - Reverse proxy API to :3000
   - Serve frontend from /var/www/socialhub/apps/web/dist
   - Enable HTTPS/SSL
   - Configure security headers

6. DATABASE
   - Create PostgreSQL database
   - Create user with minimal permissions
   - Run Prisma migrations automatically on startup

7. BACKGROUND JOBS
   - Redis queue handles post scheduling
   - Publishers push to each platform
   - Automatic retry with exponential backoff

8. MONITORING
   - PM2 for process management
   - View logs: pm2 logs
   - Status: pm2 status

9. BACKUP
   - Daily database backups to S3 or local storage
   - Keep 7-day retention policy
   - Test restore procedures

10. UPDATES
    $ git pull origin main
    $ npm install
    $ npm run build
    $ pm2 restart all
    $ npm run db:push (if schema changes)

DATABASE MODELS
===============
- User (with auth)
- Brand (per user)
- OAuthAccount (per brand per platform)
- Post (draft/scheduled/published)
- Media (with S3 keys)
- PostMedia (join table)
- Analytics (daily metrics)
